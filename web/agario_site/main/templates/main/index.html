About the Game

AgarClone is a fast-paced, realtime multiplayer game where you control a cell, eat smaller cells to grow, and avoid being eaten by larger ones. Written entirely in Python using asyncio, websockets and Pygame.

Key Features

  • Real-time multiplayer arena
  • Smooth graphics with Pygame
  • Easy to extend and customize
  • Open-source MIT-licensed
